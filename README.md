## 3. Twitter Sentiment Analysis — SVM & Random Forest
Sentiment classification on 9,820 US Airline tweets using SVM and Random Forest with TF-IDF, Word2Vec, and GloVe embeddings. TF-IDF + LinearSVM achieved the best F1-score.
Tools: Python, Scikit-learn, Gensim, NLTK, Pandas

## 4. Heart Attack Prediction — Random Forest
Binary classification model predicting heart attack risk on 1,319 patient records using Random Forest. Features include age, heart rate, blood pressure, blood sugar, CK-MB, and Troponin levels. My contribution was documenting results in a detailed analytical report.
Tools: Python, Scikit-learn, Pandas, Matplotlib, Seaborn

## 5. Breast Cancer Classification — ML (Group Project)

## 6. Stock Price Prediction — LSTM Neural Network (Group Project)
Time series prediction of Dow Jones stock prices using LSTM deep learning model. Yahoo Finance data with features including Open, High, Low, Close, and Volume. My contribution was data preprocessing analysis and documentation.
Tools: Python, TensorFlow, Keras, LSTM, Pandas, Matplotlib, Seaborn
